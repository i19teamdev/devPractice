# Gitの使い方
## ブランチの作り方・使い方
1. ` git branch -m {自分のブランチ名}`でブランチを作成し、ブランチに移動
2. ` git push -u origin {自分のブランチ名}`でリモートにブランチを追加

## add,commitについて
1. ` git add .`で全てをgitの対象に追加
2. ` git commit -m "コミットメッセージ"`で変更を保存
3. ` git push origin {自分のブランチ名}`でリモートに変更を反映させる
